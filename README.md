# NXT Travel Agency

A responsive travel website built using HTML, CSS, and JavaScript.

## Features
- Responsive design
- Karnataka tourist destinations
- Interactive booking modal
- Modern user interface
- Mobile-friendly layout
- Easy navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Destinations Included
- Coorg
- Hampi
- Malpe
- Kabini
- Mysore
- Gokarna
- Sakleshpur
- Badami
- Kollur
- Nandi Hills
